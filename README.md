# QuickCmd Vault

> Your command memory, everywhere.

QuickCmd Vault is a browser extension that helps developers store, organize and sync their most-used commands across projects and devices.

## Features

- Fast command copy
- Favorites and recent commands
- Projects and categories
- Dark / Light mode
- Local-first storage
- Chrome sync
- Cloud sync roadmap
- Import / Export JSON

## Project structure

- `extension/` browser extension source
- `docs/` project documentation
- `prompts/` prompts for Claude / Gemini / AI workflows
